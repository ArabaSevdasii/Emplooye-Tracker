# 🏢 Employee and Salary Tracking System

A highly user-friendly, modern-interface employee and salary tracking automation that I developed with AI for my dad. **Password: 1234**

## 🚀 Features

* **Daily Attendance Tracking:** Calculates hourly, half-day, or full-day deductions from Monday to Friday.
* **Automated Net Salary:** Instantly calculates deductions from the base salary and displays the net payable amount.
* **Payment History Archive:** Instead of deleting data when weekly salaries are paid, it archives them with a timestamp.
* **Soft Delete (Secure Removal):** Employees who leave the job are removed from the active list, but their past accounting records are preserved.
* **User-Friendly Interface:** Designed with massive buttons and color-coded (Red/Green) layouts specifically for elderly or non-tech-savvy tradesmen.

## 💻 Technologies Used

* **Language:** Python 3
* **Interface (GUI):** CustomTkinter
* **Database:** SQLite (Data is securely stored locally in a `.db` file)
